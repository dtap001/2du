# 2du - The best CLI TODO app

## build

```bash
npm run dev
```

## run the app itself

````bash
npm run devRun
```

## logfile location

```bash
npm run devLogs
````

## Install

```bash
npm install --global 2du
```

## CLI

```bash
2du --help

  Usage
    $ 2du

  Options
    --name  Your name

  Examples
    $ 2du --name=Jane
    Hello, Jane
```
